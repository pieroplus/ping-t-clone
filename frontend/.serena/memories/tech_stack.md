# 技術スタック

## フレームワーク・ライブラリ
- **Next.js 16** (App Router)
- **React 19**
- **TypeScript 5.9**

## スタイリング
- **Tailwind CSS 4.1**
- **PostCSS**
- **Autoprefixer**

## UIコンポーネント
- **shadcn/ui**
  - Button
  - Card
  - Input
  - Toast
  - Dropdown Menu
- **Radix UI** (shadcn/uiの基盤)
  - @radix-ui/react-dropdown-menu
  - @radix-ui/react-slot
  - @radix-ui/react-toast
- **Lucide React** (アイコン)

## ユーティリティ
- **class-variance-authority** (CVA) - バリアント管理
- **clsx** - クラス名結合
- **tailwind-merge** - Tailwindクラスのマージ

## TypeScript設定
- ターゲット: ES2017
- 厳格モード有効 (strict: true)
- パスエイリアス: `@/*` → プロジェクトルート

## 開発環境
- Node.js
- npm (パッケージマネージャー)
